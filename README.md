# Boilerplate for flask web app

## Setup & Installation

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install -r dependencies.txt
```

## Running The App

```bash
python main.py
```

## Viewing The App Localhost

Go to `http://127.0.0.1:5000`
